WildLens – Identification d'empreintes animales

Objectif du projet

Créer une application mobile permettant d'identifier des espèces animales à partir de photos d'empreintes, en utilisant un modèle de vision par ordinateur (YOLOv8).

Technologies utilisées

  • Front-end mobile : React Native
  • Back-end : Spring Boot
  • Modèle IA : YOLOv8 + OpenCV
  • API Python conteneurisée : Docker
  • Base de données : PostgreSQL (géolocalisation, date, espèce, photo)
  • Git / GitHub pour le versionning
